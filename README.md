# ffmpeg-simple-web

## 使用
```
git clone git@github.com:Djooxx/ffmpeg-simple-web.git
cd ffmpeg-simple-web
python app.py
```
访问:
```
http://localhost:5000/
```

### 1.显示视频信息
### 2.截取视频
### 3.提取视频的音频
