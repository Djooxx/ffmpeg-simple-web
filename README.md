# ffmpeg-simple-web

## 使用
```
git clone git@github.com:Djooxx/ffmpeg-simple-web.git
cd ffmpeg-simple-web
# 创建虚拟环境
python -m venv venv
# 激活虚拟环境
# Windows:
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
访问:
```
http://localhost:5000/
```

## 功能
### 1.显示视频信息
### 2.截取视频
### 3.提取视频的音频
### 4.截取音频
### 5.语音转文字(whisper)
